# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** [security@agentoffernetwork.com](mailto:security@agentoffernetwork.com)

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Response Timeline

| Action | Timeline |
|--------|----------|
| Acknowledgment | Within 2 business days |
| Initial assessment | Within 5 business days |
| Fix or mitigation | Within 30 days |

## Scope

This policy covers the SKILL.md definition and documentation in this repository. For vulnerabilities in the packaged Python MCP server or SDK, please report to the [main AON monorepo](https://gitlab.com/jolibox-dev-team/aon/aon-main).

## Coordinated Disclosure

We follow coordinated disclosure practices. Please do not publicly disclose vulnerabilities until a fix has been released.
